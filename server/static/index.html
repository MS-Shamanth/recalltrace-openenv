How It Works

🔍

The Investigator

Inspects nodes, traces lineages, and cross-references data to find contamination. Rewarded +2.0 for correct quarantines, penalized -1.5 for false positives.

🎭

The Adversary

Chooses intervention types (relabeling, mixing, deletion) and placements to maximize Investigator failure. Learns which strategies cause the most confusion.

🧠

Causal Inference

30-50% of graph edges are hidden. The agent must perform abductive reasoning to identify which hidden intervention produced the contamination pattern.

⚔️

Self-Play

The environment's difficulty is not static. The adversary adapts its curriculum based on the investigator's failure modes — creating an ever-harder challenge.

Key Themes

Theme 3.1

World Modeling

Belief state tracking with P(contaminated) per node. Agent maintains probabilistic world model and reasons under uncertainty.

Theme 4

Recursive Skill Amplification

Adversarial self-play creates an automatic difficulty curriculum. Both agents improve simultaneously in a co-evolutionary loop.
